Time

For at håndtere tid i JavaScript, kan du bruge Date objektet, som giver metoder til at arbejde med datoer og tider. Her er et eksempel på, hvordan man kan få den aktuelle dato og tid: 

  
    const now = new Date();

    console.log(now.toString()); // Viser den aktuelle dato og tid som en streng

Date objektet kan også bruges til at lave specifikke datoer og tider, og manipulere disse (f.eks. tilføje dage, måneder osv.). Det er nyttigt for en lang række opgaver, lige fra at vise datoer og tider på en webside, til at beregne tidsintervaller.

Fetch

fetch API'et i JavaScript tillader dig at foretage HTTP-forespørgsler til servere fra en webbrowser. Dette bruges ofte til at anmode om data fra en webserver og indlæse den data ind i en webapplikation uden at genindlæse siden. Her er et grundlæggende eksempel på, hvordan man bruger fetch til at anmode om data: 

  
    fetch('https://api.example.com/data')

    .then(response => {

    if (!response.ok) {

    throw new Error('Network response was not ok');

    }

    return response.json(); // Parser svaret som JSON.

    })

    .then(data => {

    console.log(data); // Arbejder med den modtagne data

    })

    .catch(error => {

    console.error('There was a problem with your fetch operation:', error);

    });

Dette eksempel viser, hvordan du kan sende en GET-forespørgsel til en URL, parse svaret som JSON, og håndtere eventuelle fejl, der måtte opstå under processen. fetch understøtter også andre HTTP-metoder som POST, PUT, DELETE osv., hvilket gør det til et kraftfuldt værktøj til at interagere med web-API'er.
